### README.md
```markdown
# Bella's Restaurant Website

Family-owned restaurant website serving Tarrytown, NY since 1979.

## Features
- Responsive design
- Online menu with shopping cart
- Image gallery
- Contact information
- Service descriptions

## Local Development

1. Clone the repository:
   ```bash
   git clone <your-repo-url>
   cd bellas-restaurant
   ```

2. Open `index.html` in your browser:
   - **Option 1**: Double-click `index.html`
   - **Option 2**: Use a local server (recommended):
     ```bash
     # Python 3
     python -m http.server 8000
     
     # Python 2
     python -m SimpleHTTPServer 8000
     
     # Node.js (if you have http-server installed)
     npx http-server
     ```
   - Then visit `http://localhost:8000`

3. Upload your restaurant images using the upload interface at the top of the page

## Deployment Options

### GitHub Pages
1. Push to GitHub
2. Go to Settings > Pages
3. Select main branch
4. Your site will be live at `https://yourusername.github.io/bellas-restaurant`

### Netlify
1. Drag and drop the entire folder to netlify.com
2. Or connect your GitHub repo for automatic deployments

### Traditional Web Hosting
1. Upload all files via FTP to your web host
2. Ensure folder structure is maintained

## IMPORTANT: Adding Your Images

The website currently has placeholder SVG images. To add your real restaurant photos:

### Step 1: Save Your Images
Save your 8 restaurant photos in the `images/` folder with these exact names:
- `breakfast-collage.jpg` - Your breakfast food collage
- `eggs.jpg` - Sunny side up eggs photo
- `breakfast-dish.jpg` - The breakfast dish with gravy
- `waffles.jpg` - Waffles with marshmallows
- `salad.jpg` - Fresh salad with strawberries
- `desserts.jpg` - Dessert collage
- `greek-salad.jpg` - Greek salad with feta
- `storefront.jpg` - Restaurant exterior

### Step 2: Update index.html
Replace the image loading script at the bottom of `index.html` with this:

```javascript
<script>
    // Load images on page load
    window.addEventListener('DOMContentLoaded', function() {
        // Gallery
        document.getElementById('gallery-img-1').src = 'images/breakfast-collage.jpg';
        document.getElementById('gallery-img-2').src = 'images/desserts.jpg';
        document.getElementById('gallery-img-3').src = 'images/storefront.jpg';
        
        // Menu items
        document.getElementById('menu-img-1').src = 'images/breakfast-dish.jpg';
        document.getElementById('menu-img-2').src = 'images/eggs.jpg';
        document.getElementById('menu-img-3').src = 'images/waffles.jpg';
